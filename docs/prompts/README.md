# Prompt Library

Reusable prompts for working with Claude on CITYARRAY.

## Session Management
- `01-session-kickoff.md` - Load context, surface risks, propose plan
- `02-session-closeout.md` - Capture state, handoff note
- `03-project-memory-update.md` - Update canonical truth

## Engineering
- `07-architecture-proposal.md` - Design new components
- `08-adr-writer.md` - Document architecture decisions
- `11-code-spec-review.md` - Review code/specs for issues
- `12-test-plan.md` - Create test strategy

## Analysis
- `10-red-team-review.md` - Attack assumptions, find weaknesses

## Usage

Replace `[PROJECT]` with `CITYARRAY` when using these prompts.

Example:
```
You are assisting with CITYARRAY.

Load and summarize:
- Project memory file(s)
- Current to-do list
...
```
