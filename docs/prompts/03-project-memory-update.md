# Project Memory Update Prompt

Update project memory for `[PROJECT]` as canonical truth.

Incorporate:
- New facts established
- Decisions made and rationale
- Assumptions and confidence
- Open questions and blockers

Rules:
- Preserve important historical decisions.
- Remove stale or disproven assumptions.
- Keep memory concise and scannable.

Output sections:
- `Current State`
- `Key Decisions`
- `Known Unknowns`
- `Next Checkpoint`
