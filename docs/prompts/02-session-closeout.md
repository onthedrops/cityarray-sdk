# Session Closeout Prompt

Close this session for `[PROJECT]`.

Produce:
1. What changed today (files, decisions, outputs).
2. What remains open (ranked by priority).
3. Risks introduced or retired.
4. Exact next-session starting point.
5. A handoff note for future sessions.

Also update:
- Project memory summary
- To-do list statuses

Output format:
- `Summary`
- `Decisions`
- `Open Items`
- `Next Actions`
