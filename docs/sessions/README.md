# Session Logs

Per-session notes for tracking what happened.

## Usage

1. Copy `SESSION_LOG_TEMPLATE.md` to `YYYY-MM-DD.md`
2. Fill in during/after session
3. Reference in `PROJECT_MEMORY.md` session history

## Template Fields

- **Objective**: What we set out to do
- **Work Completed**: What got done
- **Decisions Made**: Choices and rationale
- **Risks/Issues**: Problems encountered
- **Files Changed**: Code/config modified
- **Next Actions**: What's next

## When to Create

Create a session log when:
- Major decisions made
- Significant code changes
- Problems solved (capture for troubleshooting)
- Architecture changes
